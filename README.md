# Rust Library

![Build Status](https://github.com/rust-lang/book/workflows/CI/badge.svg)

此项目用于对Rust语言进行由浅入深的学习再到多个项目进行强化学习，他用于帮助刚入门和已经有点经验的开发者，它主要包含以下几个模块。

## 模块如下：

* basics: Rust所有基础概念
* complex: Rust复杂概念
* example: 项目例子
* leetcode: 算法
* microsoft: 微软推出的Rust教程
* web: 用于Rust语言编写的后端Web API


## 构建：

```bash
$ cargo build 
```
切换到每个模块下，可以通过test进行测试
```bash
$ cargo test 
```
